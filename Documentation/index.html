Installing Template.

  1. After unzip the document pack, open the HTML folder to find all the template files.
  2. You can view this template in any browser, and you also can edit this template without internet connection (May not working on Google Fonts).
  3. You will need to upload this files to your website using the FTP in order to use it on your website.
  4. Make sure you upload the required files/folders listed below :
    • HTML/css Extra Stylesheets folders
    • HTML/img Images folders
    • HTML/js Javascript folders
    • HTML/webfonts Webfonts (Font Awesome) folders
    • HTML/index.html Index file/Homepage
    • HTML/style.css Main stylesheet file

Google Web Fonts

By default, the template loads this font from Google Web Font Services, you can change the font with the one that suits you best.

						/*We use Nunito sans from Google fonts as Default font of this template*/		                
						@import url('https://fonts.googleapis.com/css2?family=Nunito+Sans:ital,wght@0,200;0,300;0,400;0,600;0,700;0,800;0,900;1,200;1,300;1,400;1,600;1,700;1,800;1,900&display=swap');

Remember to change the font into style.css
